# 配置仓库文件

可以叫参考一下我的然后改成你自己的   

```
|
|---posts            # 存文章
|    |---cpp       # 分类
|    |---html
|    |---hexo
|    |---教程
|
|--- 。。。。。。       # 放自己喜欢的  友链，相册，盆友圈，说说
|
|--- README.md。     # 重点，文章目录，下一章节说
|    
```
